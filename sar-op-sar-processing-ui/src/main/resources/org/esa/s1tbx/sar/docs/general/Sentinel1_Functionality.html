Sentinel-1

SENTINEL-1 Functionality

The Sentinel-1 Toolbox provide several tools specifically to support processing and analysis of Sentinel-1 products.

Reading S-1 Products

  The Toolbox supports reading of products for all modes and all product levels
  • Level-0  - Metadata only
  • Level-1 SLC/GRD
  • Level-2 OCN

Product can be opened and read directly from zip files.

Sentinel-1 Specific Tools

  • Calibration
  • Thermal Noise Removal
  • Precise Orbit Application
  • GRD and SLC Slice Assembly
  • TOPSAR Split
  • TOPSAR Deburst and Merge
  • SLC to GRD Processing
  • TOPSAR Interferometry
    • TOPSAR Backgeocoding
    • Enhanced Spectral Diversity
    • Burst-handling Interferogram and Coherence
    • Deburst and Merge of Derived Products
